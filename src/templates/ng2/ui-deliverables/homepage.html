Welcome to the NEW UI Deliverable!

I am working on this new UI Deliverables site developed with Laravel & Angular 2. It will eventually integrate the site with the new Bootstrap 4 framework (still under alpha). I'll probably wait until the final release before I implement it. I am use NPM as the package manager and GULP as the task runner. The site is developed using SASS and Typescript, that GULP compiles/transpiles to CSS and JavaScript/ES5.

I set up two GIT respositories for this application.

  1. Back End repository for the Laravel code
  2. Front End repsitory for Node, Gulp, JavaScript, Typescript, CSS, SASS, fonts and HTML templates.
